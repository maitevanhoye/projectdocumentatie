Wat heb ik gedaan?

Aan de hand van de moderne code-editor Visual Studio Code werkte ik verder aan zowel de inhoud als de opmaak van mijn webpagina. Hiervoor gebruikte ik CSS-styling die ik koppelde aan mijn HTML-structuur. Ik verwerkte feedback van medestudenten en testte verschillende stijlen en tekstaanpassingen uit.

Wat heb ik geleerd?

Ik leerde dat kleine aanpassingen in lettergrootte, kleur, etc. een grote impact kunnen hebben op de gebruikerservaring. Ook het direct kunnen bekijken van de resultaten van mijn code in de browser was erg handig om mijn ontwerp te controleren.
